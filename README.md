# SudokuSense 🧠

SudokuSense is an AI-powered Sudoku tutor that helps users learn and improve their Sudoku skills through intelligent hints, explanations, and adaptive difficulty.

## 🚀 Features (Planned)
- Sudoku puzzle generator
- Backtracking-based solver
- Explainable hint system
- Adaptive difficulty based on user performance
- Interactive web UI

## 🛠 Tech Stack
- Frontend: React
- Backend: FastAPI (Python)
- ML: scikit-learn (future)
- Database: SQLite / PostgreSQL (future)

## 📌 Project Status
🚧 In development (building core Sudoku engine)

## 🎯 Goal
To build a smart Sudoku system that not only solves puzzles but also teaches users how to solve them.
